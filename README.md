# NovaX Rivals Loader

Public RawGitHub loader for NovaX Rivals.

Use this when the repository is pushed:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/y5gc8zpdwh-droid/novax-rivals-loader/main/loader.lua"))()
```

The loader reads the manifest from `novax-rivals-modules` and runs the configured bootstrap/module entry.
